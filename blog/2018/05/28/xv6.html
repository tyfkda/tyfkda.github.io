Xv6を試してみた

2018-05-28
#os

教材用のOSでUnix V6を元にしているという、Xv6を試してみた。

  • 6.828 / Fall 2014
  • Repository: mit-pdos/xv6-public: xv6 OS

ビルド

  • Mac OSX上でVirutalBoxを動かして、ゲストOSとしてUbuntu 64bit版を使用
  • リポジトリを取得して、make
    • fs.img と xv6.img ができる

実行

  • ゲストOSにqemuを入れれば、そのまま実行できる
  • make qemu-nox

終了

  • C-a c
  • C-a x
