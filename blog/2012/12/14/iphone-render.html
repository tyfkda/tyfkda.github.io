iPhoneで自前描画

2012-12-14
#ios

UI部品を使って処理する方法はわかったけど、自前で描画したい。ん・ぱか工房のサンプルを動かしたいんだけど、プロジェクトの作り方とかがよくわかってなかった。UIViewとグラフィックの描画(1/4):初心者のためのiphone/ipadプログラミング入門を見てようやくわかった。

File > New > Project… でプロジェクトを作るときに、Xcode4.5ではView-based Applicationがないので、Single View Application を選びNextボタンを押して、Use Storyboards のチェックを外してプロジェクトを作成する。

次に File > New > File… で Objective-C class を選び、親クラスをUIViewにして作成する。ViewController.xibを選んで画面を選択して、View > Utilities > Show Identity Inspectorを選ぶ。Custom Classに自分が作ったクラスを指定する。

あとはサンプル通り。
